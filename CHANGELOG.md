## [Unreleased]
### Breaking Changes
- `Shape()`, `Group()`, and related composables: parameter `color: IsoColor` renamed to `material: MaterialData`. This is a binary-incompatible change — compiled callers using named-argument syntax (`color = ...`) must recompile and update call sites to `material = ...`. `IsoColor` still satisfies the new parameter as-is since it implements `MaterialData`.
